# python-exceptions

Python programming project covering errors, exceptions, and how to
correctly handle and raise them.

## Learning Objectives

- The difference between errors and exceptions
- What are exceptions and how to use them
- When we need to use exceptions
- How to correctly handle an exception
- The purpose of catching exceptions
- How to raise a builtin exception
- When we need to implement a clean-up action after an exception

## Requirements

- Ubuntu 20.04 LTS, Python 3.8.5
- All files end with a new line
- First line of all files: `#!/usr/bin/python3`
- Code follows pycodestyle (version 2.7.*)
- All files are executable

## Tasks

| File | Description |
| --- | --- |
| `0-safe_print_list.py` | Prints x elements of a list |
| `1-safe_print_integer.py` | Safely prints an integer |
| `2-safe_print_list_integers.py` | Prints and counts integers in a list |
| `3-safe_print_division.py` | Divides two integers with debug output |
| `4-list_division.py` | Divides two lists element by element |
| `5-raise_exception.py` | Raises a type exception |
| `6-raise_exception_msg.py` | Raises a name exception with a message |
